# Panduan Remix ExamGuard (SMPN 2 Gempol)

Halo! Jika Anda menerima link ini, berarti Anda ingin menggunakan sistem **ExamGuard** untuk sekolah Anda sendiri. Dengan melakukan **Remix**, Anda akan memiliki salinan aplikasi ini secara mandiri tanpa mengganggu data pemilik aslinya.

## Langkah-langkah Memulai:

1.  **Klik Tombol Remix:**
    Di pojok kanan atas layar Google AI Studio ini, klik tombol **"Remix"**. Ini akan menyalin seluruh kode ke akun Google Anda.

2.  **Siapkan Database Baru (PENTING):**
    Karena aplikasi ini menggunakan Firebase untuk menyimpan data ujian dan peserta, Anda harus membuat database sendiri agar data Anda tidak bercampur. 
    Ketik pesan ini ke Chat AI di sebelah kanan:
    > *"Tolong siapkan Firebase baru untuk aplikasi saya ini agar datanya terpisah dan mandiri."*
    
    Ikuti instruksi AI untuk menyetujui pembuatan project Firebase baru.

3.  **Sesuaikan Identitas Sekolah:**
    Anda bisa meminta AI untuk mengubah nama sekolah atau logo. Contoh:
    > *"Tolong ubah nama sekolah menjadi SMPN 1 Contoh dan ganti footer-nya."*

4.  **Akses Panel Admin:**
    Setelah aplikasi berjalan, buka menu **Akses Panel Admin** di bagian bawah halaman utama.
    *   **Email Admin Default:** Gunakan email Google Anda untuk login.
    *   **Password Default:** `admin123` (Anda bisa meminta AI untuk mengubahnya nanti).

5.  **Bagikan ke Siswa:**
    Gunakan link yang muncul di bagian atas (Shared App URL) untuk dibagikan ke siswa Anda.
